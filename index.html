Model Context Protocol

What is MCP and why does it matter?

MCP is the open standard that lets AI assistants like Claude securely connect to your data sources. Think of it as a universal adapter between your advertising platforms and AI.

▣

Real time Data Access

Claude reads your live campaign data directly, no exports and no copy paste.

◇

Secure by Design

OAuth authentication, no stored credentials, and access you can revoke anytime.

✦

Execute Actions

Create campaigns, update budgets, duplicate ads, and keep approvals in your hands.

claude.ai/settings/connectors

// Add to your MCP config
{
  "mcpServers": {
    "meta_ads": { "url": "https://meta.ads.mcp.example/" },
    "google_ads": { "url": "https://google.ads.mcp.example/" },
    "tiktok_ads": { "url": "https://tiktok.ads.mcp.example/" },
    "snap_ads": { "url": "https://snap.ads.mcp.example/" }
  }
}

Frequently Asked Questions

Do I need technical skills to use this?

No. If you can use Claude, you can connect an account and ask questions in plain English.

Which ad accounts can I connect?

Meta, Google, TikTok, and Snap ad accounts that you have permission to access.

What AI assistants work with this?

Claude, Claude Code, Cursor, ChatGPT, and any MCP compatible AI client.

Is my data safe?

Yes. Official APIs, OAuth authentication, and secure transport keep data protected.

Can I use this for client accounts?

Yes. Connect many ad accounts and analyze them separately or together.

Will connecting an ad account cause restrictions?

Approved API access is separate from ad policy enforcement. You still review and approve every change.
